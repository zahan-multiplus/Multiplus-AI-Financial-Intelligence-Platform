# Multiplus AI — Architecture Map Viewer

Interactive document tree viewer with search and single-doc scroll.

## Run locally

Because the viewer loads JSON via `fetch()`, serve the project over HTTP:

```bash
# From multiplus_ai_architecture_structure/
python -m http.server 8080
```

Then open: **http://localhost:8080/viewer/**

## Features

- **Search** — Filter the tree by section number or title.
- **Expand / collapse** — Click the toggle (▶) to show or hide children.
- **Single document** — Click a section node to scroll to that section in the right panel. Hash routing (`#1.2.1`) for direct links and refresh.

## Data

- `data/document_tree.json` — Full hierarchy (root + all PARTs).
- `data/section_content.json` — Section body text by id.

Generated by `generate_doc_tree_json.py` in the project root.
